## 4. Color Scheme

### Primary Colors
- **Background**: `rgba(255, 255, 255, 0.03)` - Semi-transparent white
- **Border**: `rgba(255, 255, 255, 0.1)` - Light border
- **Text Primary**: `#FFFFFF` - White
- **Text Secondary**: `rgba(255, 255, 255, 0.7)` - Semi-transparent white

### Accent Colors
- **Primary Accent**: `#FF6B35` - Orange
- **Accent Background**: `rgba(255, 107, 53, 0.2)` - Light orange
- **Gradient Start**: `#1a2332` - Dark blue
- **Gradient End**: `#0a0f1a` - Darker blue

### Semantic Colors
- **Rank Badge**: Orange circle with white number
- **Category Chip**: Light gray background with white text
- **Source Badge**: Orange background with white text
- **Hover State**: Increased opacity and color shift

## 5. Typography

### Font Sizes
- **Hero Title**: `text-4xl md:text-5xl` (36px/48px)
- **Section Title**: `text-2xl` (24px)
- **Card Title**: `text-lg` (18px)
- **Event Count**: `text-5xl` (48px)
- **Body Text**: `text-sm` (14px)
- **Metadata**: `text-xs` (12px)

### Font Weights
- **Bold**: Hero titles, event counts (700)
- **Semibold**: Section titles, card titles (600)
- **Regular**: Body text, metadata (400)

## 6. Spacing & Layout

### Grid Breakpoints
- **Mobile**: 1 column (< 640px)
- **Tablet**: 2 columns (640px - 1024px)
- **Desktop**: 3 columns (> 1024px)

### Card Dimensions
- **Height**: Auto (min 280px for grid cards)
- **Padding**: 20px internal
- **Gap**: 24px between cards (gap-6)
- **Border Radius**: 12px (rounded-xl)

### Margins
- **Section Spacing**: 48px bottom (mb-12)
- **Element Spacing**: 16px bottom (mb-4)
- **Card Group**: 24px gap (gap-6)

## Summary

The visual design follows these principles:
1. **Consistency**: Matches existing EventCard styling
2. **Clarity**: Clear hierarchy and readable typography
3. **Efficiency**: Grid layouts for quick scanning
4. **Feedback**: Loading states and hover effects
5. **Accessibility**: Semantic HTML and ARIA support
6. **Responsiveness**: Adapts to all screen sizes
7. **Performance**: Optimized animations and transitions

All components use the existing design system and integrate seamlessly with the current discover page layout.
